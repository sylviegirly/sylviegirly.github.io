Website / Software Details

  • Coding was done in Phoenix Code and is hosted on Neocities
  • Github is used as a repository (with this lovely tutorial by scumsuck how to do it)
  • Tested on Firefox at a 1920x1080 resolution
  • Primary fonts used are Free Pixel and Venice from dafont, Dosis and Atkinson Hyperlegible from Google Fonts
  • Contrast finder is used for any on hand colour contrast checking
  • Roleplay formatter for a WYSIWYG text editor to lazily copy and paste in here with full HTML formatting (for body text)
  • PNGGauntlet, Squoosh and iloveimg for image optimizing
