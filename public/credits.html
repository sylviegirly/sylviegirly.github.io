Website / Software Details

  • Coding was done in Phoenix Code and is hosted on Neocities
    • On ocassion I code on the codeit website for mobile coding + pushing
  • Github is used as a repository (with this lovely tutorial by scumsuck how to do it)
    • I also use github pages to host some iframes like my poll code and status.cafe
  • Tested on Firefox at a 1920x1080 resolution, and iPhone at a ... whatever fuckass resolutions they have
  • Title font is Venice from dafont, Dosis is used for header fonts, while Nunito, Atkinson Hyperlegible, Intel One Mono, and Open Dyslexic are used for the body fonts
  • Contrast finder is used for any on hand colour contrast checking + alt colours
  • Word HTML for a WYSIWYG text editor
  • PNGGauntlet and Squoosh for image optimizing
  • Opti-WEBP for image optimizing for OC/art galleries
